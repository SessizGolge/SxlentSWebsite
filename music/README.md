# Music Files

Add your music files to this folder. Supported formats:
- `.mp3` (MP3 Audio)
- `.m4a` (MPEG-4 Audio)
- `.wav` (WAV Audio)
- `.ogg` (Ogg Vorbis)

## How it works:

1. Upload your music files to this `/music/` folder
2. The music player will automatically detect and load all supported audio files
3. Files will be listed in the player in the order they appear
